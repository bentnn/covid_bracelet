Страница добавления новых контактов


Для добавления новых контактов загрузите, пожалуйста, файл в формате .txt

Контакты должны быть разбиты построчно, то есть в одной строке файла должна содержаться информация об одном контакте

Каждая строка файла должна быть в формате 'ИмяПользователя1 ИмяПользователя2 дд/мм/гггг чч:мм'

Пример: 'user1 user2 01/02/2021 12:30'

Контакты должны быть строго в этом формате. Даты, которым больше 2 месяцев, либо даты, которые еще не наступили, вызовут ошибку, то есть контакты из файла добавлены не будут. Если указана сегодняшняя дата, то время не должно позже, чем текущее. Это также вызовет ошибку.
